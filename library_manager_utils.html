LibraryManagerUtils¶

Overview¶

Miscellaneous functions that can be helpful during development of the CMake build.

Functions¶

List of Functions:

  • print_value()

  • print_target_properties()

print_variable¶
print_variable(<variable> [<mode>])

Prints a message giving the value of <variable> using message(<mode> ....). <mode> defaults to STATUS.

print_target_properties¶
print_target_properties(<target> [<mode>])

Prints all of the properties of <target> using message(<mode> ....). <mode> defaults to STATUS.
